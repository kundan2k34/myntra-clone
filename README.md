# Myntra Frontend Clone

A responsive e-commerce frontend project inspired by Myntra, built using **HTML, CSS, and JavaScript**. This project demonstrates core frontend development concepts such as dynamic product rendering, shopping bag functionality, responsive UI design, and local storage management.

---

## 🚀 Features

* Responsive homepage design
* Product listing section
* Add to Bag functionality
* Remove from Bag feature
* Shopping bag summary
* Dynamic price calculation
* Discount calculation
* Local Storage integration
* Clean and modern UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
